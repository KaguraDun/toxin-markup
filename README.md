## Use tips
Use  first
```bash
npm install
```
### Start
```bash
npm run start
```
### Production build
```bash
npm run build
```
### Run eslint auto fix
```bash
npm run lint:fix
```
### Run style lint auto fix
```bash
npm run stlint:fix
```
### Prettify pug files
```bash
npm run pug:fix
```
